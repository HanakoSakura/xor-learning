# XOR-Learning

一个简短的示例 · A short sample

使用PyTorch来计算逻辑门
<br>
Use PyTorch to calculate logic gates

支持六种逻辑门：
<br>
Support six kinds of logic gate

+ AND  (与门)
+ OR   (或门)
+ NAND (与非门)
+ NOR  (或非门)
+ XOR  (异或门)
+ NXOR (异或非门)

*`evil`文件夹是模型文件夹*
<br>
*The `evil` folder is the models folder.*

# 训练：(Training)

```bash 
bash train.sh
```

# 使用：(Usage)

```bash
python USE.py
```

And enter the model name.
<br>
(and,or,nand,nor,xor,nxor)
